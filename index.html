How about markdown?

JustType covers all html that can be generated from markdown. As a bonus, the default stack uses web-safe fonts:

  1. Serif
    • Georgia
    • serif
  2. Sans-serif
    • Verdana
    • sans-serif
  3. Monospace
    • Andale Mono
    • Lucida Console
    • monospace

Where's h5 and h6?

If a blog post needs an h5, I'll bet it also needs revising. If you desperately need an h6, though, it's easy to extend the style. Add to lib/style.scss, like so:

h5,
h6 {
    // your
    // style
    // here
}
